RunMate와 함께한 소중한 러닝 기록들이 모두 사라집니다.
탈퇴 전에 한 번 더 생각해보시는 것은 어떨까요?

회원탈퇴 요청 방법

RunMate 서비스 회원탈퇴는 아래 2가지 방법 중 편리한 방법을 선택하여 진행하실 수 있습니다.

1
앱 내에서 직접 탈퇴 (권장)
RunMate 앱 → 내피드 → 설정 → 계정 설정 → 회원 탈퇴
✨ 즉시 처리 가능하며 가장 빠른 방법입니다

📧 방법 2: 이메일로 탈퇴 요청

withdrawal@runmate.com

이메일 링크를 클릭하면 탈퇴 요청 양식이 자동으로 작성됩니다

이메일 탈퇴 요청 시 필요한 정보

이메일로 탈퇴 요청 시 원활한 처리를 위해 다음 정보를 포함해주세요:

1
닉네임&고유번호 또는 이메일 주소
계정 식별을 위해 필요합니다 (둘 중 하나만 기재해도 됩니다)
2
탈퇴 사유 (선택사항)
서비스 개선을 위한 참고자료로 활용됩니다

탈퇴 처리 절차

📋 처리 절차 안내

  • ✅ 앱 내 탈퇴: 즉시 처리 완료
  • ✅ 이메일 탈퇴: 요청 접수 → 계정 확인 → 탈퇴 처리 완료 안내

처리 기간: 앱 내 탈퇴는 즉시 처리, 이메일 탈퇴 요청은 영업일 기준 1-2일 내 처리됩니다.

탈퇴 시 주의사항

⚠️ 탈퇴 전 반드시 확인하세요

  • 모든 러닝 기록과 데이터가 영구적으로 삭제됩니다
  • 개인 프로필, 팔로우/팔로워, 알림 내역이 모두 삭제됩니다
  • 탈퇴 후에는 데이터 복구가 불가능합니다
  • 적립된 포인트나 리워드는 모두 소멸됩니다
  • 진행 중인 챌린지나 이벤트 참여가 취소됩니다
  • 동일한 정보로 재가입 시에도 기존 데이터는 복구되지 않습니다

삭제되는 정보

회원탈퇴 시 다음 정보들이 완전히 삭제됩니다:

  • 계정 정보: 휴대폰 번호, 이메일, 닉네임, 프로필 사진
  • 개인 프로필: 자기소개, 프로필 설정, 개인 통계
  • 러닝 데이터: 모든 운동 기록, 코스, 통계 정보
  • 소셜 기능: 팔로우/팔로워 목록, 메시지, 댓글
  • 알림 내역: 푸시 알림 설정, 알림 기록
  • 앱 사용 기록: 로그인 기록, 설정 정보

개인정보 처리 방침

개인정보 완전 삭제: 탈퇴 처리 완료 즉시 모든 개인정보가 안전하게 삭제됩니다. 단, 관련 법령에 따라 보관이 필요한 경우 해당 기간 동안만 별도 보관 후 완전 삭제됩니다.

법정 보관 의무가 있는 정보의 경우:

  • 계약 또는 청약철회 등에 관한 기록: 5년 (전자상거래법)
  • 소비자 불만 또는 분쟁처리에 관한 기록: 3년 (전자상거래법)
  • 서비스 이용 기록: 3개월 (통신비밀보호법)

탈퇴 취소

앱 내 탈퇴: 즉시 처리되므로 취소가 불가능합니다.

이메일 탈퇴: 요청 후 처리 완료 전까지는 탈퇴를 취소할 수 있습니다. 탈퇴 취소를 원하시는 경우 동일한 이메일 주소로 "탈퇴 취소 요청"이라는 제목으로 연락주시기 바랍니다.
